## भाग A — नियम

### चिह्न याद रखें
| चिह्न | मतलब |
|-------|--------|
| > | बड़ा |
| < | छोटा |
| ≥ | बड़ा या बराबर |
| ≤ | छोटा या बराबर |
| = | बराबर |

### एक statement से निष्कर्ष
**A > B** → A, B से बड़ा · B, A से छोटा  
**A ≥ B** → A > B या A = B

**मान्य निष्कर्ष (जोड़):**
- A > B और B > C → **A > C** ✓  
- A > B और B > C → **A > B** ✓ (पहले से)  
- A > B → **C > A** ✓ नहीं (C अज्ञात)

### दो statement — कौन सा निष्कर्ष follow?

**उदा:**  
P > Q, Q ≥ R  
→ P > R ✓ · P > Q ✓ · Q ≥ R ✓ · R > P ✗

### Coded (प्रतीक)
**@ = >, # = <, $ = ≥, & = ≤, % = =**

**A @ B** → A > B  
**पहले decodeAddress decode करो, फिर normal inequalities

### Priority (कभी-कभी)
**≥ और > mix:** P ≥ Q, Q > R → P > R या P = Q और Q > R

### गलती न करें
- **केवल** वही निष्कर्ष चुनो जो **100%** follow हो  
- "कभी-कभी" banking में कम — **definitely true / false** देखो

## भाग B — 25 प्रश्न

**निर्देश:** निष्कर्ष I और II में से कौन सही — (a) केवल I  (b) केवल II  (c) दोनों  (d) कोई नहीं

   **I.** A > C  **II.** C > A

   **I.** P > R  **II.** R ≥ P  

3. **M = N, N > O**  
   **I.** M > O  **II.** O > M  

4. **X < Y, Y ≤ Z**  
   **I.** X < Z  **II.** Z < X  

5. **A ≥ B, B ≥ C, C ≥ D**  
   **I.** A ≥ D  **II.** D > A  

6. **@ means >, # means <** — **P @ Q, Q @ R**  
   **I.** P @ R  **II.** R @ P  

7. **$ means ≥, & means ≤** — **A $ B, B & C** (A≥B, B≤C)  
   **I.** A ≥ C  **II.** C ≥ A  

8. **F > G, G = H**  
   **I.** F > H  **II.** H > F
