# Reasoning — Alphanumeric Series (Day 1)

**Time:** 60 min · **Part A:** 15 min · **Part B:** 30 questions · **Part C:** answers

---

## Part A — Five pattern types (learn with examples)

In banking exams, a series mixes **letters (A–Z)** and **digits (0–9)**. Find the **next term**.

### Type 1 — Alternating letter + number (+1 each step)
**A1, C3, E5, G7, ?**  
Letters: A, C, E, G → skip one → **I**  
Numbers: 1, 3, 5, 7 → **9**  
**Answer: I9**

### Type 2 — All letters only (constant gap)
**B, E, H, K, ?**  
B(+3)=E, E(+3)=H → **N** (K+3)

### Type 3 — All numbers only
**4, 9, 16, 25, ?**  
Perfect squares: 2², 3², 4² → **36** (6²)

### Type 4 — Letter gap changes, number fixed
**A2, D2, G2, J2, ?**  
Letter +3 each time: A→D→G→J→**M** → **M2**

### Type 5 — Vowel/consonant + arithmetic on numbers
**Z1, X4, V9, T16, ?**  
Letters backward skip 1: Z, X, V, T → **R**  
Numbers: 1, 4, 9, 16 squares → **25** → **R25**

### Solving steps (use every time)
1. Split **letters** and **numbers** into two mini-series.  
2. Check **+1, +2, +3** or **×2, squares, cubes** for numbers.  
3. Check **+1, +2, skip 1, skip 2** for letters (remember A=1 … Z=26 if needed).  
4. Pick the option that fits **both** parts.
