## भाग A — नियम

### मूल शब्द
| शब्द | संक्षेप |
|------|--------|
| **CP** (क्रय मूल्य) | खरीद की कीमत |
| **SP** (विक्रय मूल्य) | बेचने की कीमत |
| **Profit** | SP > CP → लाभ = SP − CP |
| **Loss** | CP > SP → हानि = CP − SP |

### प्रतिशत लाभ/हानि — हमेशा **CP** पर
**लाभ% = (लाभ / CP) × 100**  
**हानि% = (हानि / CP) × 100**

**उदा:** CP=400, SP=500 → लाभ=100 → लाभ% = 100/400×100 = **25%**

### SP से CP (या उल्टा)
| दिया | सूत्र |
|------|--------|
| CP, लाभ x% | SP = CP × (1 + x/100) |
| CP, हानि x% | SP = CP × (1 − x/100) |
| SP, लाभ x% | CP = SP / (1 + x/100) |

### Marked price (MP) और Discount
**MP** = दुकान पर लिखी कीमत  
**Discount d%** → **SP = MP × (1 − d/100)**

**उदा:** MP=1000, 20% छूट → SP = **800**

### झूठा लाभ (false weight)
वस्तु का वजन कम दिखाया — CP वास्तविक वजन पर, बेचा दिखावटी वजन पर  
*(परीक्षा में कभी-कभी — सूत्र प्रश्न से पकड़ो)*

### तेज़ ट्रिक
- **10% लाभ** → SP = 1.1 × CP  
- CP और SP दोनों **x% बढ़े** → लाभ% बदलता है (सीधे जोड़ नहीं)  
- **Equal profit% on CP:** same CP पर same % लाभ = same SP
