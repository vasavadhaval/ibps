## भाग A — नियम (2-statement)

### प्रकार
| प्रकार | उदाहरण |
|--------|---------|
| **All A are B** | सभी बैंक कर्मचारी कर्मचारी हैं |
| **No A is B** | कोई A, B नहीं |
| **Some A are B** | कुछ A, B हैं |
| **Some A are not B** | कुछ A, B नहीं हैं |

### निष्कर्ष कब **निश्चित** (definitely true)
- **All A are B + All B are C** → All A are C ✓  
- **All A are B + Some B are C** → Some A are C ✓  
- **No A is B + All B are C** → No A is C ✓  
- **Some A are B + All B are C** → Some A are C ✓  

### निष्कर्ष **गलत** (definitely false)
- **All A are B** → **No A is B** ✗  
- **No A is B** → **Some A are B** ✗  

### Only / Some not
**Only A are B** = All B are A (दिशा ध्यान!)  
**Some A are not B** ≠ Some B are not A (हमेशा नहीं)

### Venn ट्रिक (मन में)
1. पहला statement ड्रॉ करो  
2. दूसरा उस पर fit करो  
3. निष्कर्ष check — **केवल 100% वाला** चुनो

### Only conclusion types in MCQ
- All / Some / No — **Possibility** वाले Prelims में कभी-कभी — "may be true" अलग; आज **definite** only
