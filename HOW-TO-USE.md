# How to use your IBPS daily plan

## Every morning (2 minutes)

1. Open `daily/` and open **today’s date** file (example: `2026-05-20.md`).
2. Read the time budget at the top.
3. Work through sections **1 → 4** in order (unless a mock day says otherwise).

## Every night (5 minutes)

1. Mark completed tasks with `[x]`.
2. Write mistakes in your **Error Log** notebook.
3. Glance at **tomorrow’s** file so you know if it is a mock day.

## Files you need

| File | Purpose |
|------|---------|
| `ROADMAP.md` | Big picture — read once |
| `daily/YYYY-MM-DD.md` | **Your only task list for that day** |
| `daily/INDEX.md` | List of all dates |

## You do NOT

- Search for new YouTube channels
- Buy more books mid-way
- Re-plan the syllabus
- Skip mock **analysis** days

## If you miss a day

- Do **not** try to complete two days in one day.
- Open **today’s** file and follow it.
- Carry at most **English + PK** from yesterday (Quant + Reasoning are mandatory today).
