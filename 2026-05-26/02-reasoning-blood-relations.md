## भाग A — नियम

### Generation (पीढ़ी)
```
        दादा-दादी
       /          \
    पिता -------- माता
       \          /
         तुम (मैं)
```

### मुख्य संबंध
| कथन | संबंध |
|------|--------|
| A का पुत्र B | B, A का बेटा |
| A की बेटी B | B, A की daughter |
| A का भाई B | B, A का brother (same generation) |
| A की बहन B | sister |
| A का पिता B | B, A का father (ऊपर पीढ़ी) |
| A की माता B | mother |
| A का पति B | B husband of A |
| A की पत्नी B | wife |

### Gender clues (परीक्षा)
- **बेटा, पिता, भाई, चाचा, नाना** → male  
- **बेटी, माता, बहन, चाची, नानी** → female  
- **बहन का भाई** = आप या sibling

### कोड शब्द
| शब्द | अर्थ |
|------|------|
| Maternal | माँ की तरफ (मामा, नाना) |
| Paternal | पिता की तरफ (चाचा, दादा) |
| Nephew | भाई/बहन का बेटा |
| Niece | भाई/बहन की बेटी |
| Cousin | चाचा/मामा के बच्चे (same generation) |
| Uncle | चाचा/मामा |
| Aunt | चाची/मामी |

### ट्रिक
1. **अपने को "मैं"** रखो — बीच में  
2. **+1 generation** = ऊपर (parent) · **−1** = नीचे (child)  
3. **सास-ससुर** = पति/पत्नी के माता-पिता
