## भाग A — पाठ

### DBMS क्या है?
**Database Management System** — data **organized** store, **secure**, **query** (SQL), **multiple users**.

**उदा:** Core Banking — accounts, transactions tables

### File system vs DBMS
| | File | DBMS |
|---|------|------|
| Data | Scattered files | Structured tables |
| Query | Manual | SQL |
| Redundancy | High | Controlled |
| Security | Weak | Users, roles |

### Key terms
| शब्द | मतलब |
|------|--------|
| **Database** | related data collection |
| **Table/Relation** | rows + columns |
| **Row/Tuple** | one record |
| **Column/Attribute** | field |
| **Primary Key (PK)** | unique row identifier |
| **Foreign Key (FK)** | links to another table's PK |
| **Schema** | database structure design |

### DBMS functions
- **DDL** — structure (CREATE, ALTER, DROP)  
- **DML** — data (SELECT, INSERT, UPDATE, DELETE)  
- **DCL** — rights (GRANT, REVOKE)  
- **TCL** — transactions (COMMIT, ROLLBACK)

### Users
| Role | काम |
|------|-----|
| **DBA** | admin, backup, security |
| **Application** | banking software |
| **End user** | teller queries (limited) |

### Advantages
Data independence, less redundancy, integrity, concurrent access, backup-recovery

### बैंक
**CBS** — customer accounts, loans, transactions — **Oracle, DB2, MySQL** etc. behind
