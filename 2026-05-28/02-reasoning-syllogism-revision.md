## भाग A — Quick Revision

- All A are B
- No A is B
- Some A are B
- Some A are not B

**Golden rules:**
- All A are B + All B are C → All A are C
- Some A are B + All B are C → Some A are C
- No A is B + Some B are C → Some C are not A
