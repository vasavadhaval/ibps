## भाग A — नियम

### अनुपात (Ratio)
**a : b** = a/b (समान इकाई होनी चाहिए)

**उदा:** 2 : 5 = 2/5 · Rs 40 और Rs 50 का ratio **4 : 5** (पहले समान करो)

### समानुपात (Proportion)
**a : b :: c : d**  →  a/b = c/d  →  **a×d = b×c** (cross multiply)

**उदा:** 3 : 4 :: 15 : x  →  3x = 60  →  x = **20**

### साझा करना (divide in ratio)
Rs 1200 को 2 : 3 : 5 में बाँटो → कुल भाग 10 → 1 भाग = 120  
→ 240, 360, 600

### पार्टनरशिप (लाभ बाँट)
| प्रकार | नियम |
|--------|------|
| समय समान | पूँजी के ratio में लाभ |
| पूँजी समान | समय के ratio में |
| दोनों अलग | पूँजी×समय का ratio |

**उदा:** A Rs 5000 (12 माह), B Rs 3000 (8 माह)  
→ A:B = 5000×12 : 3000×8 = 60000 : 24000 = **5 : 2**

### मिश्रण (Alligation — सरल)
दो मिश्रण m₁ : m₂ और दरें d₁, d₂, मिश्र दर dₘ:

```
(d₂ − dₘ) : (dₘ − d₁) = m₁ : m₂
```

**उदा:** 20% और 50% मिलाकर 30% — (50−30):(30−20) = 20:10 = **2:1**

### तेज़ ट्रिक
- Ratio दोनों तरफ ×/÷ same number → ratio same  
- **a:b = 3:4** और total 35 → 7 भाग × 5 = 35, a=15, b=20  
- **Componendo:** (a+b):(a−b) परीक्षा में कभी-कभी
